Hacettepe Üniversitesi Eğitim Fakültesi BÖTE Bölümü WEB programcılığı Dersi Final Ödevi React ile API Kullanımı Konu React ile API kullanarak dinamik bir web sayfası geliştirme Amaç Öğrencilerin React bileşen yapısını, fetch veya axios ile API’den veri çekmeyi ve bu verileri kullanıcıya anlamlı şekilde sunmayı öğrenmeniz. Görev Bir web sayfası geliştiriniz. Bu sayfa, seçtiğiniz bir açık API’den (örneğin: JSONPlaceholder, OpenWeather, The Cat API, CollectAPI Spor API, TheSports API gibi) veri çekecek ve kullanıcıya liste, kart veya tablo şeklinde gösterecektir. İstemler (Yönergeler) • React projesi başlatın. • En az 3 farklı bileşen oluşturun: o Header: Sayfanın başlığı ve kısa açıklama o Content: API’den gelen verilerin listelendiği bölüm o Footer: Basit telif veya iletişim bilgisi • Kullanıcıdan istem için giriş isteyin. Mesela hava durumunda şehir bilgisi gibi. • API’den veri çekmek için useEffect ve useState hook’larını kullanın. • Kullanıcıya verileri dinamik olarak gösterin. Hava soğuksa karlı bir fon, yağmurlu ise yağmurlu bir fon gibi. (örneğin: hava durumu, rastgele fotoğraflar, kullanıcı listesi). • En az bir stil dosyası (CSS) ekleyin veya styled-components kullanın. • Kullanıcı deneyimini artırmak için: o Yükleniyor (Loading...) mesajı o Hata durumunda (Bir hata oluştu) mesajı (Yanlış şehir adı girildiyse mesela) • Bonus: Kullanıcıdan input alarak API’ye parametre gönderebilirsiniz (örneğin şehir adı girilince hava durumu getirme). Teslim Koşulları • Proje GitHub’a yüklenmeli. • README.md dosyasında: o Kullanılan API’nin linki o Projenin nasıl çalıştırılacağı o Ekran görüntüsü Son 2 hafta ödev teslimleri Lab’da bana gösterilerek yapılacaktır.
